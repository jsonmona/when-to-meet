# 언제만나? - 약속 잡기 서비스

## 소개
### 문제상황
> A: "17일 괜찮으신가요?"  
> B: "아, 저 그날은 다른 일정이 있어요."  
> A: "그럼 22일은 다들 어떠신가요?"  
> C: "제가 그때는 지방에 내려가 있을거라서 힘들어요."  

### 해결책
> A: "여기 접속해서 각자 가능한 날짜 투표해주세요."  
> B: "투표 다 했어요!"  
> C: "저도 투표했어요!"  

### 소개글
"언제만나?"는 여러명이 모인 그룹에서 약속을 잡을 때 모두가 가능한 날짜를 고르는 것을 도와주는 서비스입니다.
주최자가 날짜와 투표 종류를 설정하여 링크를 공유하면, 참여자들은 해당 링크에 접속하여 본인의 가능한 일정을 투표할 수 있습니다.
복잡한 날짜 조율을 단순화하여 빠르고 편리하게 약속을 확정해 보세요.

## 특징
### 약속 설정
- 카테고리 중심의 시간 선택: 구체적인 시간(1시, 2시) 대신 '점심', '저녁' 등 직관적인 카테고리로 일정을 조율합니다.
- 올인원 투표 시스템: 여기 접속해서 투표하는 김에 어디서 만날지나 뭘 먹을지도 한번에 투표할 수 있습니다.

## 실행 방법
이 레포지토리를 클론하고 도커를 이용해 실행합니다.
```
$ git clone https://github.com/jsonmona/when-to-meet
$ cd when-to-meet
$ docker-compose up -d
```

첫 실행시 DB 세팅이 필요합니다.
```
$ docker exec -it when-to-meet-app bash
container# npx prisma migrate deploy
```
