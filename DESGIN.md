# 언제만나? - 약속시간 잡는 서비스

## 다른 서비스와의 차별점
- 시간을 정하지 말고 카테고리를 정하자 (점심/저녁 등)
- 한 화면에서 장소 투표까지 가능

## 화면 구성

### 메인 화면
- "지금까지 전국에서 N개의 약속을 잡았어요" 같은 홍보성 정보창
- 새 약속 만들기 버튼
- 홍보용 화면 스크린샷

### 새 약속 화면
- 제목 설정
- 약속 기간 범위 설정 (예: 12/1 ~ 12/5)
- 활성화된 카테고리 설정 (점심/오후/저녁/밥/게임 등 태그 시스템)

### 약속 화면
- 약속 제목, 기간 표시
- 공유 가능한 링크 복사버튼
- 최선의 날짜 보기 버튼 (카테고리별로 최선의 날짜 표시)

#### 캘린더
- 해당 기간에 대한 캘린더 표시
- 캘린더 각 칸에는 같은 태그로 가능하다고 표시한 참여자의 최대 비율 표시 (백그라운드 색)
- 읽기모드에서 날짜 클릭시 해당 날짜에 각 참여자별 가능여부 표시
- 수정모드에서 날짜 클릭시 해당 날짜의 선택한 참여자를 선택한 태그로 표시함
- 실시간 동기화 (예정)

#### 참여자 목록
- 참여자 목록 표시
- 참여자를 추가하거나 삭제 가능
- 참여자 옆의 수정 버튼을 누르면 태그 목록을 선택하는 체크박스가 뜨고 수정모드 진입
- 실시간 동기화 (예정)
